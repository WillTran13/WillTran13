### Featured Project

#### [Automated QA Hardware Log Parser](https://github.com/WillTran13/qa_log_parser)
*A Python CLI tool and CI/CD pipeline built to automate hardware troubleshooting.*

* **The Problem:** Manually scrolling through hundreds of lines of terminal logs to find a single latency spike or sensor timeout is tedious and prone to human error.
* **The Solution:** I wrote a diagnostic script that uses regex to catch critical failures instantly. I then wrapped it in a Pytest suite and a GitHub Actions pipeline so that any future updates to the script are automatically tested before deployment.
* **Key Skills:** Python Automation, Pytest, GitHub CI/CD, Log Analysis.
